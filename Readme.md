# INFIX to POSTFIX Converter & Evaluator (C++)

A simple C++ program that converts an **infix expression** (e.g., `3+(4*5)-2^2`) into **postfix** form and then evaluates it.

## Features
- Supports `+ - * / ^` operators
- Handles parentheses
- Correct operator precedence & associativity
- Evaluates postfix expression using stack
